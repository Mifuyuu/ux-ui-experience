Usage Examples

Get started in minutes with simple configuration

Installation

# Quick Start (Recommended)
npx ux-ui-experience@latest

# Global Installation
npm install -g ux-ui-experience

Configuration

// opencode.json
{
  "mcp": {
    "ux-ui-experience": {
      "type": "local",
      "command": ["npx", "-y", "ux-ui-experience@latest"],
      "enabled": true
    }
  }
}
